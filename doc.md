# ytnobody/revealup

---

## How to use

1. Write the slide in markdown.
2. Upload the slide to internet in accessible publicly.
3. Access to [manager](/manager) and fill-in the URL for the slide.
4. Submit the form. Then, access to [Slide Page](/).
